# Someone is sus
Someone Is Sus is a curated application designed to report and store player profiles having suspicious behavior in gaming.


stack:
 - Go
 - sqlc
 - Sqlite

## local run

```shell
go mod tidy
set -a; source .env; go run .
```

## Application structure/layout

This application follows the "Organizing a Go module" guide: https://go.dev/doc/modules/layout
